Javascript Lab - Fall 2015

Instructions

  1. Create a dog class that has the following:
    • Properties
      • id
      • name
      • gender
      • breed(s)
      • age (human years)
    • Methods
      • speak - returns a string of the dogs name
      • dogYears - return age in dog years
  2. Write a program that instantiates and initializes the properties through constructor arguments. Then call both methods and write result to the web page (index.html).
  3. Concurrently create a JSON that holds the values for 3 dogs objects

Use document.getElementById('dog-name').innerHTML = your_output; to write the dogs name to the page.

Use document.getElementById('dog-years').innerHTML = your_output; to write the dogs age in dog years to the page.
